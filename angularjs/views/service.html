Давайте посмотрим, как это работает.

В первую очередь мы должны добавить angular-resource.js файл для нашего приложения

Наш сервис определен в scripts/services/recordServices.js:

angular.module("recordServices", ["ngResource"]).

factory("Record", function($resource) {

return $resource("data/:id.json", {}, {

getAll: {method: "GET", params: {id: "all"}, isArray: true}

});

});

Обратите внимание, что используется модуль API для регистрации пользовательского сервиса. Для этого мы передали имя сервиса - "Record" - в функцию фабрику.

Record-сервис зависит от $resource сервиса.

Мы также определили метод getAll, который вернет нам все записи.

Так же существует ряд стандартных методов, которые доступны по умолчанию:

{

"get": {method:"GET"},

"save": {method:"POST"},

"query": {method:"GET", isArray:true},

"remove": {method:"DELETE"},

"delete": {method:"DELETE"}

};

Теперь

у нас есть готовый сервис, и мы готовы использовать его.

Не забудьте объявить наш сервис-модуль в приложении (смотрите /scripts/app.js):

angular.module("angular-example", ["recordServices"])

Теперь вам достаточно просто указать имя сервиса в параметрах контроллера, и Angular заинжектит его для вас:

app.ServiceCtrl = function ($scope, Record, $modal) {


$scope.records = Record.getAll();


$scope.showRecord = function (id) {

Record.get({id: id}, function(result) {

$scope.chosen = result;

...

});

};

};

Здесь мы использовали наш сервис дважды: первый раз мы вызвали нами определенный метод getAll чтобы получить список всех записей, и затем мы использовали метод по умолчанию get, чтобы подучить детали каждой записи.
