Ok, let's start

Our application starts in /index.html document.

If you open it, you'll notice the following code sample:

<html lang="en" ng-app="oms">

ng-app is a directive, which switches Angular JS framework ON. In our case, Angular JS will be active within whole document, though we are free to put this directive to some definite portion of page.

oms is just a name of our module, which will be activated. You can find it, if open /app.js file:

angular.module('oms', [])

If you look through the

/index.html

you'll notice, that there are only footer and header there but no content.

That's because Angular JS have taken cared of it. All the content is represented with a single tag:

<div ng-view></div>

ng-view tells our module, that we will be displaying all our content within marked tags.
